# CipherSQLStudio

CipherSQLStudio is a browser-based SQL learning platform where users can practice SQL queries against pre-configured assignments.

## Features
- Assignment listing with difficulty levels
- SQL editor with query execution
- Intelligent hints using LLM (Gemini)
- Real-time feedback and error handling
- Mobile-first responsive UI

## Tech Stack
- Frontend: React, Vite, SCSS
- Backend: Node.js, Express
- Database: PostgreSQL (sandbox), MongoDB
- LLM: Google Gemini (Free Tier)
